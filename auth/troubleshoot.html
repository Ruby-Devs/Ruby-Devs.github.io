Troubleshooting Ruby Auth

If you are having issues with using RubyAuth, this page may help you.

Common Issues

These are some common issues:

  • Commenting on the wrong profile
  • Bad internet connection or none at all
  • Ruby Auth site has been updated. Fixable by hitting Ctrl + F5 to reload without cache
  • Scratch servers may be down
